# WanandoidCompose版本

#### 介绍
此Wanandroid app客户端项目使用Android最新技术Jetpack Compose完成，遵循MVVM架构思路，以下为本项目用到的框架：
compose
viewModel+liveData
retrofit+okhttp3
coroutine以及其flow框架
paging3
room
accompanist
hilt
